# kanban-flaskapp
A web-based personalized Kanban (board) made using flask

```app structure
app/
├── __init__.py
├── main_app.py
├── models.py
│
├── templates/
│   ├── base.html
│   ├── board.html
│   ├── login.html
│   └── signup.html
│ 
└── static/
    └── css/
        └── styles.css
```
# Virtual Env and Installations

commands:

    python3.6 -m venv .venv
    source .venv/bin/activate
    pip3 install -r requirements.txt
